Making a pack:

	1. Use the json to set the information for your skin pack. Comments for each field are below:
		- GameTileVersion: (int only) Use the value from the current tile version at the time of making the pack.
		- Name: The name of the pack.
		- Description: The description you want to appear.
		- Animations: (true or false only!) does the pack use the animation system?
		- ResourceNames: The name of each resource.
		- ResourceColors: The color of each resource.

	2. Create a image and call it "Icon.png" this is used for the pack icon.

	3. wip

Important Info:
	
	1. The game will not check and warn about any errors!
